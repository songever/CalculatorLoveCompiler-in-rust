# CalculatorLoveCompiler-in-rust

简要介绍
本项目是一个用 Rust 实现的简单示例编译器/解释器（CalculatorLoveCompiler），用于展示如何在 Rust 中构建基础的词法、语法分析与执行流水线。旨在作为学习与演示用途，不依赖复杂外部工具。 

特别感谢 VectorizeOrz 视频分享给予的灵感  
https://www.bilibili.com/video/BV1G346zHEDz/?spm_id_from=333.1365.list.card_archive.click&vd_source=7c611e4b4450383775c4f29754e5ead1  

运行前提
- 已安装 Rust（包含 cargo）。如果尚未安装，请参考：https://www.rust-lang.org/tools/install

快速运行
1. 克隆或进入本项目目录：
   cd /root/rust/vectorize-in-rust/CalculatorLoveCompiler-in-rust

2. 使用 cargo 运行项目：
   cargo run

备注
- 以上命令会编译并运行当前仓库的默认二进制。开发时可用 cargo run --release 进行 Release 模式构建以获得更好的性能。
- 项目主要用于学习目的，具体用法和示例请参考代码中的注释与测试用例。
